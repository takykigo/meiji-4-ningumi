◆ 1. 神宮 (Jingu)

「神宮」を名乗る社

皇室の祖先神・天皇・皇族と深い縁を持つ神を祀る最高位の社号です。現在、全国に約25社存在します。

● 代表例

伊勢神宮 / 明治神宮 / 熱田神宮 / 鹿島神宮 / 香取神宮 / 北海道神宮 / 橿原神宮 / 平安神宮 / 宇佐神宮 / 宮崎神宮 / 霧島神宮 / 伊弉諾神宮 など
  • 平安期に神宮と呼ばれたのは伊勢・香取・鹿島の3社のみ。
  • 明治以降、勅許を得て「神宮」へ改称する社が増加。
  • 戦後は神社本庁の承認で名乗る例（北海道神宮等）もある。
◆ 2. 大社 (Taisha)

「大社」を名乗る社

古代の社格で高位にあった神社。出雲大社が起源となり、後に有力神社へ広がりました。全国に24社あります。

● 代表例

出雲大社 / 春日大社 / 諏訪大社 / 熊野大社 / 住吉大社
※氣比神宮（神宮）との混同に注意
  • 元来は「出雲大社」だけを指す特別な呼称。
  • 戦後に多くの有力神社が「大社」を名乗るようになった。
◆ 3 & 4. 社号の意味

肩書きの分類

【神社】

最も一般的。特別な基準はなく、全国の大多数がこれに該当します。

【宮 (ぐう)】

皇子・皇孫など皇族ゆかりの人物を祀る。天満宮や東照宮もこの分類です。

【大社・神宮】

前述の通り、由緒や神格が特に高い社に許される名称です。

◆ 5. 序列の仕組み

一宮・二宮・三宮

これは「名前」ではなく、旧国地域における「格式の順位」を示す呼び名です。

● 一宮： 真清田神社(尾張) / 氣比神宮(越前) / 田村神社(讃岐)

● 二宮・三宮： 地域ごとに異なり、一宮に次ぐ格式。

なぜ序列があるのか？
古代、地方に赴任した政治家（国司）が最初に参拝する順序が「一宮」だったため。政治と祭祀の中心を示す制度でした。

💡 最重要ポイント

神宮

皇室ゆかりの神を祀る。全国約25社。伊勢は別格。

大社

出雲大社が起源。古代の高位神社。全国24社。

一宮〜三宮

地域の参拝順。社号ではなく「序列」を指す。
